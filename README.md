## 🚀 Vercel da Deploy Qilish

### 1-usul: GitHub orqali (tavsiya etiladi)
1. GitHub da yangi repository yarating: `sarvar-website`
2. Fayllarni yuklang (quyidagi ko'rsatmalar)
3. [vercel.com](https://vercel.com) ga kiring
4. "Add New Project" → GitHub repository ni tanlang
5. "Deploy" tugmasini bosing — tayyor! ✅

### 2-usul: Vercel CLI orqali
```bash
npm install -g vercel
cd sarvar-website
vercel
```
